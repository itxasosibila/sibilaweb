# Sibila Aroma - V1 (Astro Base)

Diese Version enthält die fertig codierte Astro-Webseite mit den finalen, echten Texten von Itxaso. Alles Unnötige (Word-Dokumente, Notizen, Skripte) wurde aus diesem Ordner herausgehalten.

## Anleitung für den Live-Gang
1. Lade den kompletten Inhalt **dieses Ordners** (`V1_Astro_Base`) in dein neues GitHub Repository hoch.
2. Melde dich bei Netlify an und klicke auf "Import from GitHub".
3. Wähle das Repository aus. Netlify wird sofort erkennen, dass es ein Astro-Projekt ist.
4. Klicke auf "Deploy" – und fertig! Die Seite ist innerhalb weniger Sekunden unter der Netlify-Domain (und nach deinem DNS-Update auch unter der echten `.es` Domain) live.
